# What is Pypass?

**Pypass** is a simple FOSS python application that stores, and generates new passwords for a user. 

# Benefits of Pypass:

1. User only needs a **single master password** to access all their other passwords.

2. It is totally **free of cost** and **open source** under the **MIT License** on **GitHub** so, people can see how the program **works**, **contribute** to **adding features** and **improving** the app and **fork** it and make it their **own**.

3. It has a **local-first** approach and stores the passwords in a **.csv** file.
